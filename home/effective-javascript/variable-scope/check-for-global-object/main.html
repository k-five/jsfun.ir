check-for-global-object

In some cases we have to check for the presence of an object in global scope; e.g. JSON.

We know that the global object in javascript (in browsers) is window.

also we know that this in global context refers to window object.

Thus

// check if this equals window
this === window // true

So if we want to check for the existence of JSON we can:

// directly using window object
if( window.JSON ){
    // yes we have it
} else {
    // an alternative
}

Since this refers to window, we also can:

// directly this instead of window
if( this.JSON ){
    // yes we have it
} else {
    // an alternative
}

Also we have in keyword in javascript which can be used to check if an object has a property or not.

So we could:

// directly using window object
if( "JSON" in window ){
    // yes we have it
} else {
    // an alternative
}

or using this

// if we are sure this is global object
if( "JSON" in this ){
    // yes we have it
} else {
    // an alternative
}
