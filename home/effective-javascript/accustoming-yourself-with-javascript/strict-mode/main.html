how to use "use strict" when we have more than one files

"use strict" is an ECMAScript Standard to force web browser Compatibility.

  • Within a file it can be applied only at the top most of the file
  • and within a function the same rule - at the top most of a function

If we have two files and one of them uses [strict mode] but the other not, bundling these two files will remove the [strict mode] effect if the rule has not been put at the top most of the bundled file.

File-1.js does not use "use strict"


// file-1.js
// no strict mode
function(){
	// ...
}

file-2.js uses "use strict"


"use strict";
// file-2.js
function(){
	// ...
}

Bundling these two files has no effect!


// bundling file-1.js && file-2.js
// strict mode has no effect
// file-1.js
function(){
	// ...
}
"use strict";
// file-2.js
function(){
	// ...
}

Solution is to use IIFE (= Immediately Invoke Function Expression). 


// solution for each file
(function(){
	"use strict";
	function(){
	// ...
}
}());

NOTE: If you use Webpack it automatically adds "use strict" to the bundled file.
